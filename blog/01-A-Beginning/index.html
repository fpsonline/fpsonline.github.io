Project Details

This is an FPS game project called Blaze Battlegrounds, using mainly the Ursira library - one of the libraries that supports making games in the Python programming language. This project serves the Open Source Software Development course of Saigon University.

Our game supports Windows and MacOS platforms. Players can play together by using the same network source, such as players connecting to the same Wifi network...
